<objective>
Add the raw/display activity score split to InstrumentAnalysis, create the DisambiguationState interface and factory, create instrumentFamilies.ts constants, and wire the score split into AnalysisTick.ts so that downstream code reads displayActivityScore while internal analysis uses rawActivityScore.

Purpose: This is the architectural keystone for all disambiguation work. Without the raw/display split, disambiguators would cascade-suppress each other. Without DisambiguationState, disambiguators have nowhere to store stateful buffers. Without instrumentFamilies.ts, disambiguators cannot detect which pairs are present.

Output: Modified types.ts, InstrumentActivityScorer.ts, AnalysisTick.ts; new instrumentFamilies.ts
</objective>
