  <name>Task 1: Create NodeLayout.ts and NodeAnimState.ts</name>
  <files>src/canvas/nodes/NodeLayout.ts, src/canvas/nodes/NodeAnimState.ts</files>
  <action>
Create `src/canvas/nodes/` directory with two files:

**NodeLayout.ts:**
- Export `NodePosition` interface with `x: number` and `y: number` (fractional [0,1] of logical canvas dims)
- Export `computeNodePositions(count: 2 | 3 | 4): NodePosition[]`
  - count=2: horizontal `[{x:0.30, y:0.50}, {x:0.70, y:0.50}]`
  - count=3: triangle `[{x:0.50, y:0.25}, {x:0.28, y:0.68}, {x:0.72, y:0.68}]`
  - count=4: diamond `[{x:0.50, y:0.20}, {x:0.22, y:0.50}, {x:0.78, y:0.50}, {x:0.50, y:0.80}]`
- Export `INSTRUMENT_ORDER` as `['guitar', 'drums', 'keyboard', 'bass']` — maps to diamond positions [top, left, right, bottom]. This places bass (bottom) and drums (left) adjacent for the Phase 6 pocket line.

**NodeAnimState.ts:**
- Export `RippleState` interface: `startMs: number`, `durationMs: number`, `maxRadius: number`, `color: string`, `baseX: number`, `baseY: number`
- Export `NodeAnimState` interface with all fields from RESEARCH.md Pattern 1: `baseRadius`, `currentRadius`, `glowIntensity`, `glowCanvas` (HTMLCanvasElement), `radiusNudge`, `ripples` (RippleState[]), `orbitAngle`, `breathePhase`, `lastSeenBassOnsetSec`, `lastSeenDrumOnsetSec`, `lastSeenBeatCounter`, `lastSeenDownbeatSec`, `lastPocketScore` (for glow re-creation threshold gate)
- Export `createNodeAnimState(color: string, baseRadius: number): NodeAnimState` — creates the plain object with all defaults (-1 for lastSeen timestamps, empty ripples array). Calls `createGlowLayer(baseRadius * 2, color)` for the glowCanvas.
- Export `lerp(a: number, b: number, t: number): number` — standard linear interpolation
- Export `lerpExp(current: number, target: number, factor: number, deltaMs: number): number` — frame-rate-independent exponential lerp using `1 - Math.pow(1 - factor, deltaMs / 16.667)` as the t value
- Export `drawAndUpdateRipple(ctx: CanvasRenderingContext2D, ripple: RippleState, nowMs: number): boolean` — draws a single ripple ring and returns true if still alive. Compute `elapsed = nowMs - ripple.startMs`, if `>= durationMs` return false. `progress = elapsed / durationMs`, `radius = maxRadius * progress`, `alpha = 1 - progress`. Draw with `ctx.save()`, `ctx.globalAlpha = alpha`, `ctx.strokeStyle = ripple.color`, `ctx.lineWidth = 2`, arc stroke, `ctx.restore()`.
- Export `updateRipples(ctx: CanvasRenderingContext2D, ripples: RippleState[], nowMs: number): void` — iterates backward over ripples array, calls drawAndUpdateRipple, splices expired entries.

NOTE: Ripple utilities are placed here (not in drawGlow.ts) so both bass (05-03) and drums (05-04) animations can import them from a Wave 1 artifact, enabling 05-03 and 05-04 to run in parallel.
