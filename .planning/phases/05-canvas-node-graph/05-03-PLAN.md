Create `src/canvas/nodes/drawGlow.ts`:

1. **Export color constants** (pre-parsed RGB for zero-alloc interpolation):
   ```
   AMBER_RGB = { r: 0xb4, g: 0x53, b: 0x09 }  // #b45309
   BLUE_RGB = { r: 0x1e, g: 0x40, b: 0xaf }    // #1e40af
   ```

2. **Export `pocketToGlowColor(pocketScore: number): string`:**
   - Clamp pocketScore to [0,1]
   - Interpolate each RGB channel: `lerp(BLUE_RGB.r, AMBER_RGB.r, t)` etc.
   - Return `rgb(r,g,b)` string
   - Import `lerp` from `./NodeAnimState`

3. **Export `drawGlow(ctx, glowCanvas, cx, cy, intensity): void`:**
   - If `intensity <= 0`, return early
   - `const size = glowCanvas.width` (radius*4 from createGlowLayer)
   - `ctx.save()`
   - `ctx.globalAlpha = intensity`
   - `ctx.globalCompositeOperation = 'lighter'` (additive blend for glow)
   - `ctx.drawImage(glowCanvas, cx - size/2, cy - size/2, size, size)`
   - `ctx.restore()`

4. **Export `drawAndUpdateRipple(ctx, ripple, nowMs): boolean`:**
   - Compute `elapsed = nowMs - ripple.startMs`
   - If `elapsed >= ripple.durationMs`, return false (expired)
   - `progress = elapsed / ripple.durationMs`
   - `radius = ripple.maxRadius * progress`
   - `alpha = 1 - progress`
   - Draw: `ctx.save()`, `ctx.globalAlpha = alpha`, `ctx.strokeStyle = ripple.color`, `ctx.lineWidth = 2`, `ctx.beginPath()`, `ctx.arc(ripple.baseX, ripple.baseY, radius, 0, Math.PI * 2)`, `ctx.stroke()`, `ctx.restore()`
   - Return true (still alive)

5. **Export `updateRipples(ctx, ripples, nowMs): void`:**
   - Iterate backward, call `drawAndUpdateRipple` for each, `splice(i, 1)` if expired

2. **Add `updateBassBreath` function** (private method or inline in render):
   - If `beat.bpm` is null: return 0.15 (static low glow)
   - Compute `beatPeriodMs = (60 / beat.bpm) * 1000`
   - Advance `animState.breathePhase = (animState.breathePhase + deltaMs / beatPeriodMs) % 1.0`
   - Compute sine: `const sine = Math.sin(animState.breathePhase * Math.PI * 2)`
   - Map to [0.2, 0.8]: `const breathe = 0.2 + (sine + 1) / 2 * 0.6`
   - Modulate by pocket score: `return breathe * (0.5 + beat.pocketScore * 0.5)`

3. **In the instrument rendering loop**, when instrument is 'bass':
   - **Onset detection:** Compare `beat.lastBassOnsetSec` to `animState.lastSeenBassOnsetSec`. If different:
     - Update `animState.lastSeenBassOnsetSec`
     - Set `animState.glowIntensity = 1.0` (flash)
     - Push a new ripple if `animState.ripples.length < 4`: `{ startMs: performance.now(), durationMs: 800, maxRadius: 80, color: 'rgba(180,83,9,0.6)', baseX: x, baseY: y }`
   - **Breathing:** Call `updateBassBreath` to get breathe intensity. Use `Math.max(breatheIntensity, animState.glowIntensity)` as final glow intensity (onset flash overrides breathing).
   - **Glow intensity decay:** `animState.glowIntensity = lerpExp(animState.glowIntensity, 0, 0.05, deltaMs)` — slow decay after onset flash
   - **Pocket-score glow color shift (VIZ-05):** Check `Math.abs(beat.pocketScore - animState.lastPocketScore) > 0.05`. If threshold crossed, re-create glow canvas: `animState.glowCanvas = createGlowLayer(animState.baseRadius * 2, pocketToGlowColor(beat.pocketScore))`.    - **Draw ripples:** Call `updateRipples(ctx, animState.ripples, performance.now())` AFTER drawing the node circle

4. **Guard all beat reads:** If `state.beat` is null, skip all beat-dependent animation (bass glow defaults to static 0.15 intensity).
