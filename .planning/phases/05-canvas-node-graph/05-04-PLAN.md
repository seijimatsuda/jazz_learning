Add drums-specific animation logic to CanvasRenderer render loop. All logic goes in the instrument rendering loop when instrument is 'drums':

**1. Beat Nudge (VIZ-06):**
- Each frame, decay radiusNudge: `animState.radiusNudge = lerpExp(animState.radiusNudge, 0, 0.92, deltaMs)`. Snap to 0 when below 0.5px: `if (animState.radiusNudge < 0.5) animState.radiusNudge = 0`
- On new drum onset detected: set `animState.radiusNudge = 6` (sharp +6px)
- The currentRadius lerp from 05-02 already includes `animState.radiusNudge` in its target: ensure the line reads `animState.currentRadius = lerpExp(animState.currentRadius, animState.baseRadius + animState.radiusNudge, 0.15, deltaMs)`. If 05-02 didn't include radiusNudge in the target, add it now.

**2. Onset Detection (use timestamps, NOT beatCounter):**
- Compare `beat.lastDrumOnsetSec !== animState.lastSeenDrumOnsetSec`
- If new onset: update `animState.lastSeenDrumOnsetSec`, set nudge, spawn ripple
- Guard: only fire if `beat.bpm !== null` (suppress during rubato)

**3. Crisp Ripple on Beat (VIZ-07):**
- On new drum onset: push ripple to animState.ripples (if length < 4):
  `{ startMs: performance.now(), durationMs: 300, maxRadius: 60, color: '#e0f2fe', baseX: x, baseY: y }`
- Use `updateRipples(ctx, animState.ripples, performance.now())` from drawGlow.ts to draw and expire ripples

**4. Downbeat Double Ripple (VIZ-08):**
- Compare `beat.lastDownbeatSec !== animState.lastSeenDownbeatSec`
- If new downbeat AND `beat.bpm !== null`: update lastSeenDownbeatSec, push SECOND ripple with `durationMs: 500`, `maxRadius: 90`, `color: '#e0f2fe'` (larger, slower fade)
- The first ripple from step 3 fires on the same frame (downbeat is also a beat onset)

**5. Timing Offset Orbit (VIZ-09):**
- Constants: `ORBIT_THRESHOLD_MS = 30`, `ORBIT_RADIUS_PX = 3`, `ORBIT_SPEED = 0.004` (radians/ms)
- If `Math.abs(beat.timingOffsetMs) > ORBIT_THRESHOLD_MS`:
  - Advance `animState.orbitAngle = (animState.orbitAngle + ORBIT_SPEED * deltaMs) % (Math.PI * 2)`
  - Compute offset: `ox = Math.cos(animState.orbitAngle) * ORBIT_RADIUS_PX`, `oy = Math.sin(...) * ORBIT_RADIUS_PX`
- Else: `animState.orbitAngle = 0`, `ox = 0`, `oy = 0`
- Apply orbit offset to drawing position: draw the drums node at `(x + ox, y + oy)` instead of `(x, y)`
- Update ripple baseX/baseY to use the orbit-offset position for newly spawned ripples

**6. Draw order for drums:** glow (if any) -> node circle -> ripples (drawn on top)

All animations MUST use deltaMs for frame-rate independence (iOS 30fps compatibility).
