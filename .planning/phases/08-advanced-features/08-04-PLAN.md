**Extend Timeline.tsx with annotation capability:**

1. **Add state for annotation input:**
   ```typescript
   const [annotationInput, setAnnotationInput] = useState<{ timeSec: number; leftPct: number } | null>(null);
   const [annotationText, setAnnotationText] = useState('');
   const annotations = useAppStore(s => s.annotations);
   const addAnnotation = useAppStore(s => s.addAnnotation);
   ```

2. **Modify click handler (the existing handleClick or onClick on the bar):**
   Currently the click handler seeks to the clicked timestamp. Modify to check for Shift key:
   - If `e.shiftKey` is true AND `isFileLoaded`:
     - Calculate timeSec from click position (same math as existing seek)
     - Calculate leftPct for positioning the input overlay
     - Set `annotationInput = { timeSec, leftPct }`
     - Set `annotationText = ''`
     - Do NOT seek (prevent the default seek behavior)
   - If `e.shiftKey` is false:
     - Existing seek behavior unchanged

3. **Render annotation input overlay (when annotationInput is not null):**
   ```tsx
   {annotationInput && (
     <div
       style={{
         position: 'absolute',
         left: `${annotationInput.leftPct}%`,
         top: '-40px',
         transform: 'translateX(-50%)',
         zIndex: 10,
       }}
       className="bg-[#1a1a2e] border border-amber-500/50 rounded px-2 py-1 flex gap-1"
     >
       <input
         type="text"
         value={annotationText}
         onChange={(e) => setAnnotationText(e.target.value)}
         onKeyDown={(e) => {
           if (e.key === 'Enter' && annotationText.trim()) {
             addAnnotation(annotationInput.timeSec, annotationText.trim());
             setAnnotationInput(null);
             setAnnotationText('');
           }
           if (e.key === 'Escape') {
             setAnnotationInput(null);
             setAnnotationText('');
           }
         }}
         autoFocus
         placeholder="Add note..."
         className="bg-transparent text-white text-xs outline-none w-32"
       />
       <button
         onClick={() => {
           if (annotationText.trim()) {
             addAnnotation(annotationInput.timeSec, annotationText.trim());
             setAnnotationInput(null);
             setAnnotationText('');
           }
         }}
         className="text-amber-400 text-xs hover:text-amber-300"
       >
         +
       </button>
     </div>
   )}
   ```

4. **Render annotation markers on the timeline bar:**
   Inside the timeline bar container (the same div that has the progress fill and beat grid), add:
   ```tsx
   {annotations.map((ann) => {
     const pct = duration > 0 ? (ann.timeSec / duration) * 100 : 0;
     return (
       <div
         key={ann.id}
         title={ann.text}
         style={{
           position: 'absolute',
           left: `${pct}%`,
           top: 0,
           bottom: 0,
           width: '3px',
           backgroundColor: '#f59e0b',  // amber
           opacity: 0.8,
           cursor: 'pointer',
           zIndex: 2,
         }}
         onClick={(e) => {
           e.stopPropagation();  // don't trigger seek or annotation
           // Optional: show the annotation text in a small popover
         }}
       />
     );
   })}
   ```

5. **Add Shift+click hint text** below the timeline (small muted text): "Shift+click to annotate" — only show when isFileLoaded is true and no annotation input is active.
