**Create src/audio/PitchDetector.ts:**

Implement the ACF2+ autocorrelation pitch detector. This module is pure functions operating on pre-allocated buffers.

```typescript
// Core function: detectPitch(buf: Float32Array, sampleRate: number, correlationBuffer: Float32Array): number
// 1. RMS gate: compute RMS of buf. If RMS < 0.01, return -1 (no pitch in silence/noise).
// 2. Autocorrelation: for i=0..SIZE-1, for j=0..SIZE-i-1, correlationBuffer[i] += buf[j]*buf[j+i]
//    IMPORTANT: Zero the correlationBuffer before use (loop to set all to 0, not .fill(0) which may allocate).
// 3. Find first dip: walk correlationBuffer while c[d] > c[d+1].
// 4. Find first peak after dip: scan from d to SIZE, track maxVal and maxPos.
// 5. Parabolic interpolation for sub-sample accuracy.
// 6. Return sampleRate / T0 (Hz).
//
// NOTE: Use the passed-in correlationBuffer (pre-allocated on PitchAnalysisState) — do NOT allocate new Float32Array.
```

Helper function: `pitchesMatch(a: number, b: number): boolean`
- Returns true if two pitches are within 50 cents (ratio < 1.029).
- Returns false if either is <= 0.

Factory function: `initInstrumentPitchState(fftSize: number): InstrumentPitchState`
- Pre-allocates correlationBuffer as Float32Array(fftSize).
- Sets pitchHz=-1, prevPitchHz=-1, pitchFrameCount=0, isMelodic=false.

Update function: `updatePitchState(pitchState: InstrumentPitchState, buf: Float32Array, sampleRate: number): void`
- Calls detectPitch with pitchState.correlationBuffer.
- If detected pitch matches prevPitchHz (within 50 cents), increment pitchFrameCount.
- If pitchFrameCount >= 3, set isMelodic = true, stablePitchHz = pitchHz.
- If pitch doesn't match or is -1, reset pitchFrameCount to 0, isMelodic = false.
- Update prevPitchHz = pitchHz.
