  <name>Task 2: Create InstrumentActivityScorer with band mapping, scoring, history, and init</name>
  <files>

  <action>
    Create src/audio/InstrumentActivityScorer.ts with the following exports:

    **InstrumentName type:**
    ```typescript
    export type InstrumentName = 'bass' | 'drums' | 'keyboard' | 'guitar';
    ```

    **INSTRUMENT_BAND_MAP constant:**
    Maps instrument name to FrequencyBand names it owns:
    - bass: ['bass'] (20–250Hz)
    - drums: ['drums_low', 'drums_high', 'ride'] (60–300Hz + 2–8kHz + 6–10kHz)
    - keyboard: ['mid'] (250–2000Hz)
    - guitar: ['mid_high'] (300–3000Hz)

    **resolveBandsForInstrument(name: InstrumentName, lineup: InstrumentName[]): string[]**
    - If both keyboard AND guitar are in lineup, return INSTRUMENT_BAND_MAP[name] as-is
    - If only ONE mid-range instrument (keyboard or guitar, not both) is in lineup, and `name` is that instrument, return ['mid', 'mid_high'] (full mid-range, INST-05)
    - For bass and drums, always return INSTRUMENT_BAND_MAP[name] regardless of lineup

    **computeActivityScore(freqData, bandNames, bands, calibration, prevScore, smoothingAlpha = 0.7): number**
    - For each bandName in bandNames:
      - Find the matching FrequencyBand from bands
      - Find the matching CalibrationThresholds from calibration
      - If band not found or cal not found or cal.peak === 0, skip
      - Compute: energy = getBandEnergy(freqData, band) / cal.peak (normalize to peak)
    - Average all normalized energies; clamp to [0, 1]
    - Apply EMA smoothing: result = prevScore * (1 - smoothingAlpha) + raw * smoothingAlpha
    - Return the smoothed result

    **HISTORY_LENGTH constant = 100**

    **pushHistory(instr: InstrumentAnalysis, score: number): void**
    - Write score into instr.historyBuffer at instr.historyHead
    - Advance historyHead: (historyHead + 1) % HISTORY_LENGTH
    - Increment historySamples if < HISTORY_LENGTH

    **initAnalysisState(lineup: InstrumentName[], fftSize: number): AnalysisState**
    - Creates InstrumentAnalysis for each instrument in lineup:
      - bandNames from resolveBandsForInstrument(name, lineup)
      - activityScore: 0, role: 'silent', roleSinceSec: 0
      - historyBuffer: new Float32Array(HISTORY_LENGTH)
      - historyHead: 0, historySamples: 0
      - timeInRole: { soloing: 0, comping: 0, holding: 0, silent: 0 }
    - Creates edgeWeights: Record<string, number> for all instrument pairs (alphabetically sorted key), initialized to 0
    - Pre-allocates: prevRawFreqData = new Uint8Array(fftSize / 2), rawTimeDataFloat = new Float32Array(fftSize)
    - Returns full AnalysisState with isAnalysisActive: false, lastAnalysisMs: 0

    CRITICAL: No new Float32Array or Uint8Array allocations should EVER happen outside initAnalysisState. All buffers are allocated once here and reused.

    Import getBandEnergy from './FrequencyBandSplitter'.
    Import types from './types'.
  </action>
