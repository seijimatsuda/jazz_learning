**Create drawPocketLine.ts** implementing EDGE-01 through EDGE-06:

Import lerpExp and lerp from `../nodes/NodeAnimState`, drawGlow from `../nodes/drawGlow`, EdgeAnimState from `./EdgeAnimState`.

**Exported function:**
```
drawPocketLine(
  ctx: CanvasRenderingContext2D,
  bassX: number, bassY: number, bassRadius: number,
  drumsX: number, drumsY: number, drumsRadius: number,
  animState: EdgeAnimState,
  pocketScore: number,
  lastSyncEventSec: number,
  deltaMs: number,
): void
```

**Endpoint termination:** Compute direction vector from bass to drums. Subtract `bassRadius` from bass end and `drumsRadius` from drums end along the direction vector so the line terminates at each node's circumference, not center.

**Pocket state machine:**
- `pocketScore > 0.7` = 'tight': lineWidth=4, color='#4ade80', draw with flowing dashes using `ctx.setLineDash([12, 8])` and `ctx.lineDashOffset = -animState.dashOffset`. Increment `animState.dashOffset = (animState.dashOffset + deltaMs * 0.06) % 20`.
- `pocketScore > 0.4` = 'loose': lineWidth=2.5, color='#fde68a', draw with `ctx.quadraticCurveTo` using perpendicular normal wobble. Control point = midpoint + perpNormal * 8 * sin(animState.wobblePhase). Increment `animState.wobblePhase = (animState.wobblePhase + deltaMs * 0.003) % (Math.PI * 2)`. Use `ctx.setLineDash([])` explicitly.
- else = 'free': lineWidth=1.5, color='#94a3b8', draw plain `ctx.moveTo/lineTo`. Use `ctx.setLineDash([])`.

**CRITICAL:** Always wrap in `ctx.save()`/`ctx.restore()` for lineDash state isolation (iOS Safari).

**EDGE-05 sync flash:** Compare `lastSyncEventSec` to `animState.lastSeenSyncEventSec`. If changed AND > 0, set `animState.flashIntensity = 1.0` and update `animState.lastSeenSyncEventSec`. If `animState.flashIntensity > 0.01`, call `drawGlow(ctx, animState.flashGlowCanvas, midX, midY, animState.flashIntensity)` at the line midpoint. Decay: `animState.flashIntensity = lerpExp(animState.flashIntensity, 0, 0.08, deltaMs)`. Snap to 0 below 0.02.

**EDGE-06 floating label:** After drawing the line, render text above the midpoint.
- `getPocketLabel(pocketScore)`: >0.7 "deep in the pocket", >0.5 "locked in", >0.3 "swinging loose", else "playing free"
- Draw at (midX, midY - 14) with `ctx.textAlign='center'`, `ctx.textBaseline='bottom'`, font `'11px sans-serif'`, fillStyle `'rgba(255,255,255,0.6)'`

**EDGE-01:** The pocket line is ALWAYS visible — never check weight/opacity. Always draw regardless of pocketScore value.
