  <action>
Create `drawCommunicationEdges.ts` implementing EDGE-07 and EDGE-08.

Import: lerpExp from `../nodes/NodeAnimState`, EdgeAnimState from `./EdgeAnimState`, EDGE_TYPE, EDGE_COLOR from `./edgeTypes`, INSTRUMENT_ORDER from `../nodes/NodeLayout`.

**Exported function:**
```
drawCommunicationEdges(
  ctx: CanvasRenderingContext2D,
  nodePositions: { x: number; y: number }[],
  nodeRadii: number[],
  edgeAnimStates: Record<string, EdgeAnimState>,
  edgeWeights: Record<string, number>,
  canvasW: number,
  canvasH: number,
  deltaMs: number,
): void
```

**Internal constants:**
- `PAIRS`: all 5 non-pocket pairs as [indexA, indexB, key] tuples. Pre-compute from INSTRUMENT_ORDER at module load:
  - bass_guitar, bass_keyboard, drums_guitar, drums_keyboard, guitar_keyboard
  - Skip 'bass_drums' — that's the pocket line (Plan 01)

**Per-pair logic (iterate PAIRS):**

1. **Read raw weight:** `const rawWeight = edgeWeights[key] ?? 0;`

2. **Smooth weight:** `animState.currentWeight = lerpExp(animState.currentWeight, rawWeight, 0.12, deltaMs);`

3. **Determine visual state from smoothed weight:**
   - `< 0.3` = 'hidden': targetOpacity = 0
   - `0.3-0.4` = 'static_thin': lineWidth = 1.5, targetOpacity = 0.4
   - `0.4-0.7` = 'subtle': lineWidth = 3, targetOpacity = 0.65
   - `>= 0.7` = 'animated': lineWidth = 5, targetOpacity = 0.9

4. **Smooth opacity:** `animState.currentOpacity = lerpExp(animState.currentOpacity, targetOpacity, 0.1, deltaMs);`

5. **Early return:** if `animState.currentOpacity < 0.01`, skip drawing this edge entirely (no stroke for hidden edges).

6. **Compute endpoints:** Use `nodePositions[idxA].x * canvasW`, etc. Terminate at node circumference by subtracting `nodeRadii[idx] * directionVector` from each endpoint (same pattern as drawPocketLine).

7. **Determine base color:** Look up `EDGE_TYPE[key]` to get edge type, then `EDGE_COLOR[type]` for RGB. Convert to `rgb(r,g,b)` string. (Tension tinting will be added in Plan 03 — for now use the base color directly.)

8. **Draw:**
   - Wrap in `ctx.save()` / `ctx.restore()` always
   - Set `ctx.globalAlpha = animState.currentOpacity`
   - Set `ctx.strokeStyle = colorString`
   - Set `ctx.lineWidth = lineWidth`
   - If 'animated' state (>= 0.7): use `ctx.setLineDash([12, 8])` with `ctx.lineDashOffset = -animState.dashOffset`. Increment dashOffset: `animState.dashOffset = (animState.dashOffset + deltaMs * 0.04) % 20`. (Slightly slower than pocket line — 0.04 vs 0.06.)
   - If 'subtle' or 'static_thin': `ctx.setLineDash([])`, plain `moveTo/lineTo`
   - `ctx.beginPath(); ctx.moveTo(x1,y1); ctx.lineTo(x2,y2); ctx.stroke();`

No wobble on communication edges — wobble is pocket-line-only.
