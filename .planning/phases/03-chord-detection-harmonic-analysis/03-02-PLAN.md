
    **TENSION_TARGETS (exported):**
    ```typescript
    export const TENSION_TARGETS: Record<ChordFunction, [number, number]> = {
      tonic:       [0.0,  0.2 ],
      subdominant: [0.2,  0.45],
      dominant:    [0.55, 0.75],
      altered:     [0.75, 1.0 ],
    };
    ```

    **LERP_RATE constant = 0.05**
    **HISTORY_CAP constant = 32** (3 seconds at 10fps = 30 + 2 margin)
    **GHOST_OFFSET constant = 30** (30 ticks = 3 seconds at 10fps)

    **initTensionState(): TensionState (exported)**
    ```typescript
    return {
      currentTension: 0,
      tensionHistory: new Float32Array(HISTORY_CAP),
      tensionHistoryHead: 0,
      tensionHistorySamples: 0,
    };
    ```

    **updateTension(tension: TensionState, chordFunction: ChordFunction): void (exported)**
    1. Compute target = midpoint of TENSION_TARGETS[chordFunction]: `(lo + hi) / 2`
    2. Lerp: `tension.currentTension += (target - tension.currentTension) * LERP_RATE`
    3. Clamp to [0.0, 1.0]: `tension.currentTension = Math.max(0, Math.min(1, tension.currentTension))`
    4. Write to ring buffer:
       ```
       tension.tensionHistory[tension.tensionHistoryHead] = tension.currentTension;
       tension.tensionHistoryHead = (tension.tensionHistoryHead + 1) % HISTORY_CAP;
       if (tension.tensionHistorySamples < HISTORY_CAP) tension.tensionHistorySamples++;
       ```

    **getGhostTension(tension: TensionState): number (exported)**
    - If tensionHistorySamples < GHOST_OFFSET, return 0 (not enough history yet)
    - Read value from 30 ticks ago:
      `const idx = (tension.tensionHistoryHead - GHOST_OFFSET + HISTORY_CAP) % HISTORY_CAP;`
      `return tension.tensionHistory[idx];`

    File header: explain chord function to tension mapping, lerp rate behavior (0.0 to 1.0 in ~3s), and ghost line concept.

    CRITICAL: No allocations in updateTension or getGhostTension. Only initTensionState allocates.
