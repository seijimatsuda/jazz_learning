<objective>
Create the TensionMeter Canvas component that renders a vertical gradient bar with the current tension level and a ghost line showing the tension from 3 seconds ago, then integrate it into the CanvasRenderer's rAF loop.

Purpose: The tension meter is the primary visual indicator of harmonic tension. The gradient (blue to red) gives musicians an immediate, intuitive read of harmonic movement. The ghost line provides temporal context -- "where was the tension 3 seconds ago" -- creating a sense of direction (rising vs falling tension).

Output: TensionMeter.ts with gradient rendering and ghost line. CanvasRenderer.ts integration.
</objective>
