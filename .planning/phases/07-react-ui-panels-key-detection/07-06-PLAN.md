**Polling (500ms / 2fps):**
Use `setInterval(500)` to poll `audioStateRef.current.chord?.chordLog`.

On each poll:
1. Snapshot the chordLog: `const snapshot = [...(audioStateRef.current.chord?.chordLog ?? [])]`.
2. Get current playback time from `getCurrentPosition(audioCtx, transport)`.
3. Call `detectKey(snapshot, currentTimeSec)` to get the detected key.
4. Push detected key to Zustand: `useAppStore.getState().setDetectedKey(result.key, result.mode)`.
5. Map each snapshot entry to display data:
   - Look up `CHORD_TEMPLATES[entry.chordIdx]` for `.root`, `.type`
   - Build chord display name (e.g. 'Cmaj7', 'G7', 'Am7')
   - If detected key is available, call `chordFunctionInKey(root, type, keyRoot, keyMode)` to get function label
   - Otherwise, use the generic chord function from `CHORD_TEMPLATES[entry.chordIdx].function` with the existing plain English labels
   - Compute tensionLevel: tonic=0.1, subdominant=0.35, dominant=0.65, altered=0.85
6. Set `logEntries` with the mapped data (most recent first — reverse the array).

**Tension color coding (UI-11):**
Function to map tension level to background color:
```typescript
function tensionBgColor(tension: number): string {
  if (tension < 0.3) return 'rgba(74,222,128,0.15)';   // green tint
  if (tension < 0.6) return 'rgba(251,191,36,0.15)';    // amber tint
  if (tension < 0.85) return 'rgba(251,146,60,0.15)';   // orange tint
  return 'rgba(248,113,113,0.15)';                       // red tint
}
```

**Format helper:**
```typescript
function formatTimestamp(sec: number): string {
  const m = Math.floor(sec / 60);
  const s = sec % 60;
  return `${m}:${s.toFixed(1).padStart(4, '0')}`;
}
```

**Chord display name helper:**
```typescript
const TYPE_SUFFIX: Record<string, string> = {
  major: '', minor: 'm', maj7: 'maj7', m7: 'm7', dom7: '7', dim7: 'dim7', m7b5: 'm7b5', alt: 'alt',
};
function chordDisplayName(root: string, type: string): string {
  return root + (TYPE_SUFFIX[type] ?? type);
}
```
