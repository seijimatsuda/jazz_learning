**Create BandSetupPanel.tsx:**

A React component that renders the band lineup configuration panel.

```typescript
import { useAppStore } from '../store/useAppStore';

const AVAILABLE_INSTRUMENTS = ['keyboard', 'bass', 'drums', 'guitar'] as const;

const INSTRUMENT_ICONS: Record<string, string> = {
  keyboard: '\u{1F3B9}', // piano emoji
  bass: '\u{1F3B8}',     // guitar emoji (closest to bass)
  drums: '\u{1F941}',    // drum emoji
  guitar: '\u{1F3B5}',   // musical note emoji
};

const BAND_LABELS: Record<string, string> = {
  keyboard: '250-2000 Hz',
  bass: '20-250 Hz',
  drums: '60-300 Hz + 2-8 kHz',
  guitar: '300-3000 Hz',
};
```

The component:
1. Reads `lineup` and `isFileLoaded` from Zustand.
2. Renders a dropdown (`<select>`) at the top that lists instruments NOT already in the lineup. When an instrument is selected, it's added to the lineup via `setLineup([...lineup, instrument])`. The dropdown resets to a placeholder "Add instrument..." after selection.
3. Below the dropdown, renders a list of instrument rows. Each row shows:
   - Icon (from INSTRUMENT_ICONS)
   - Name (capitalized)
   - Frequency band label (from BAND_LABELS)
   - Remove button (X) that filters the instrument out of lineup
4. When `isFileLoaded` is true:
   - The dropdown is disabled with a tooltip/title "Load a new file to change lineup"
   - Remove buttons are disabled (gray, no cursor pointer)
   - A small "Locked" badge appears at the top

Style: dark card matching existing app style — bg #13131f, border rgba(99,102,241,0.3), text white, accent violet.
