**Method 1: drawBpmDisplay**
```typescript
private drawBpmDisplay(ctx: CanvasRenderingContext2D, x: number, y: number, bpm: number | null): void
```
- Position: bottom-left corner. Call with `x = 20, y = h - 20`.
- Draws text: `"quarter note = {bpm}"` or `"quarter note = --"` when bpm is null.
- Use the Unicode quarter note character: `'\u2669'` (this is the standard musical quarter note symbol).
- Font: `'bold 16px sans-serif'`
- Color: `'rgba(243,244,246,0.8)'` (near-white, slightly transparent) when bpm is not null; `'rgba(107,114,128,0.5)'` (gray, dim) when null.
- Text alignment: `ctx.textAlign = 'left'`, `ctx.textBaseline = 'bottom'`.
- Format: `ctx.fillText(`\u2669 = ${bpm !== null ? bpm : '\u2014'}`, x, y)`

**Method 2: drawRoleLegend**
```typescript
private drawRoleLegend(ctx: CanvasRenderingContext2D, x: number, y: number): void
```
- Position: top-left corner. Call with `x = 16, y = 20`.
- Draws 4 rows vertically spaced 18px apart.
- Each row: a small filled circle (radius 5) + text label 14px to the right.
- Roles and colors (matching VIZ-12 / InstrumentRoleOverlay):
  - 'soloing' → '#f59e0b' (amber)
  - 'comping' → '#3b82f6' (blue)
  - 'holding' → '#6b7280' (gray)
  - 'silent' → '#374151' (dark gray)
- Font: `'11px sans-serif'`
- Text color: `'rgba(255,255,255,0.5)'`
- Text alignment: `ctx.textAlign = 'left'`, `ctx.textBaseline = 'middle'`.
- For each role, draw: `ctx.arc(x, y + i*18, 5, 0, Math.PI*2)` with fillStyle set to role color, then `ctx.fillText(roleName, x + 14, y + i*18)`.

**Integration in render():**
After the tension meter block (`if (tension) { ... }`), before `// -- Schedule next frame`:


IMPORTANT: Wrap both calls in `ctx.save()` / `ctx.restore()` to isolate text alignment state from the rest of the render loop. Or use save/restore inside each method.
